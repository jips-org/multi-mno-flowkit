FlowKit setup with multiple data providers.

A demonstration of FlowKit software used by a Non-Governmental Organisation with multiple Mobile Network Operators providing Call Detail Record.

View on GitHub Download .zip Download .tar.gz

Welcome to this demonstration of FlowKit software used by an NGO with multiple data providers.

This project is a collaboration of JIPS, FlowMinder and Alsenet in order to demonstrate possible ways to combine aggregated de-anonymised data from multiple data providers such as CDR data provided by multiple MNOs.
